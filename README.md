## üöÄ Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- No installation required - runs directly in the browser!

### Installation
1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/meme-generator.git
   cd meme-generator
   ```

2. **Open in browser**
   ```bash
   # Simply open index.html in your web browser
   open index.html
   # Or use a local server (recommended)
   python -m http.server 8000
   ```

3. **Start creating memes!**

## üì± How to Use

1. **Select a Template**
   - Browse through the template gallery on the left
   - Use the category filter to find specific types
   - Click on any template to select it

2. **Add & Customize Text**
   - Click "Add Text" to create new text elements
   - Drag text around to position it perfectly
   - Use the right sidebar to customize:
     - Text content
     - Font family and size
     - Colors and stroke
     - Bold/italic formatting

3. **Download & Share**
   - Click "Download" to save as PNG
   - Use "Share" to post on social media
   - "Clear All" to start over
